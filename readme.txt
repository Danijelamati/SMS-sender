react-native android app that informs patients about their appointment

app scans events from google calendar and takes phone numbers from the description

app runs in the background(headlessjs) and checks new events in the calendar and sends SMS from it

app can send notification SMS from the picked start date to end date

app has a function to not send SMS on the current day

user can choose the time and customize the message

every patient in the message can get his unique appointment time
